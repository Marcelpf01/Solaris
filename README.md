## 🚀 Overview

**Solaris** is a modern, immersive web experience designed to bring the Solar System to life through cinematic visuals, smooth animations, and scientifically inspired storytelling.

Every planet features a unique landing page with custom AI-generated imagery, responsive layouts, interactive galleries, and engaging educational content.

The project combines premium UI/UX design with high-performance frontend engineering to create an experience that feels closer to a digital space documentary than a traditional website.

# 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/solaris.git
```

Navigate into the project

```bash
cd solaris
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
